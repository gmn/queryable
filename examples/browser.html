Enter some words, then close your browser and re-open it again. Navigate to this page.

This data is stored locally in a variable provided by most browsers: localStorage
queryable.js is fetched from the server in a <script src="/queryable.js"></script> tag.
When it is loaded with queryable.open("DatabaseName"); it looks to see if a db with a matching name is found in localStorage.
If it is, it loads it.
